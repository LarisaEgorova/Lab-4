# Лабораторная работа №4 — задача Дирихле для уравнения Пуассона

**Вариант 2.** Область `[-1,1] × [-1,1]`. Методы: верхней релаксации (МВР/SOR) и Зейделя.

Вычислительное ядро — на **C++**, графический интерфейс, таблицы и графики — на **Python** (tkinter + matplotlib).

---

## Что считается

Уравнение `Δu = -f` с граничными условиями Дирихле, дискретизация 5-точечной
разностной схемой 2-го порядка, решение СЛАУ итерационными методами.

**Тестовая задача** (точное решение известно):
- `u*(x,y) = exp(1 - x² - y²)`
- `f*(x,y) = 4(1 - x² - y²)·exp(1 - x² - y²)` (равно `-Δu*`)
- погрешность `ε1 = max|u* - v(N)|`

**Основная задача** (точное решение неизвестно):
- `f(x,y) = |sin³(πxy)|`, `μ1=μ2=1-y²`, `μ3=μ4=|sin(πx)|`
- точность `ε2` по правилу Рунге (сетки `(n,m)` и `(2n,2m)`)

Требуемая точность: `ε = 0.5·10⁻⁵`.

1. Расширения VS Code: **C/C++** и **Python**.

## Замечания о сходимости

- **Тестовая задача** (гладкое `u*`): порядок схемы **≈ 2**, отношение
  погрешностей при удвоении сетки → **4**.
- **Основная задача** (негладкая `f = |sin³(πxy)|`): порядок **≈ 1**,
  отношение → **2** — теорема требует гладкости, а модуль синуса даёт изломы.
- **МВР vs Зейдель:** при оптимальном ω МВР сходится в десятки раз быстрее
  по числу итераций при той же точности.
